# Spec 0030 — 3D state map (E10.5)

- **Epic / Issue**: E10.5 / #53
- **Status**: Implemented
- **Author**: agent
- **Needs human decision?**: no — MapLibre + deck.gl ColumnLayer extrusions over a
  **demo fixture** of state centroids + metrics (no public geo API yet). Live contract/
  anomaly aggregates are a follow-up.

## 1. Problem

SYSTEM_DESIGN §4.12 calls for **3D maps** (MapLibre + deck.gl) with state extrusions by
contract volume / anomaly density. The web has Explore/Graph but no spatial surface.

## 2. Scope & non-scope

- **In scope**
  - Route `/map` + nav link.
  - MapLibre basemap (free style, no API key) + deck.gl **ColumnLayer** extrusions at
    state/FCT centroids.
  - Metric toggle: `contract_volume` | `anomaly_density`.
  - Demo fixture: 36 states + FCT with lat/lng + both metrics; labelled demo banner.
  - Click column → detail panel (state name, metrics).
  - Theme-aware basemap (light/dark CARTO-style free tiles).
  - Tests: fixture coverage, metric selector helper, `/map` route (map canvas mocked).
- **Out of scope**
  - Full ADM1 polygon GeoJSON extrusion (follow-up if we vendor simplified polygons).
  - Live aggregates from Postgres/Memgraph.
  - Time animation, clustering, heatmap layers.
  - Offline tile packs.

## 3. Design

```text
/map → MapLibre + Deck overlay → ColumnLayer(height ∝ metric)
     → click → detail; toggle metric
```

## 4. Data contracts

```ts
type StateMetric = {
  id: string;           // e.g. "LA"
  name: string;         // "Lagos"
  lat: number;
  lng: number;
  contract_volume: number;  // demo NGN units
  anomaly_density: number;  // 0–1 demo score
};

type MapMetric = "contract_volume" | "anomaly_density";
```

## 5. Acceptance criteria (testable)

- [x] `/map` shows title, demo banner, map region, metric toggle.
- [x] Fixture includes ≥37 jurisdictions (36 states + FCT).
- [x] Toggling metric changes which field drives extrusion height (pure helper tested).
- [x] Selecting a state shows name + both metrics in a detail panel.
- [x] Nav includes Map.
- [x] `pnpm --filter @naijaledger/web lint typecheck test` pass.
- [x] No Mapbox token / paid tile dependency.

## 6. Risks & mitigations

- **Demo mistaken for live volumes** — persistent banner.
- **WebGL in jsdom** — mock map/deck in route tests.
- **Bundle size** — accept MapLibre/deck.gl as design-required deps.

## 7. Open questions

None blocking. Polygon GeoJSON + live aggregates tracked as follow-ups.
